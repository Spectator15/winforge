# WinForge v2.0 - Windows Toolkit
Made by Danish

## Features
- System Repair (DISM, SFC, CHKDSK)
- Cleanup (Temp, Update Cache, DNS, Network Reset)
- Dependencies (.NET 6/7/8/9, VC++ Redists, DirectX, WebView2)
- Tweaks (Essential, Advanced/Caution, Preferences) with tooltips and presets
- Debloat (remove pre-installed apps with confirmation and restore point)
- Windows Updates (Default, Security Only, Disable All)
- System Tools (Legacy Windows Panels)
- Restore Point (manual creation)

## How to run
Download WinForge.exe from the Actions tab artifacts. Double-click. Click Yes for admin rights.
