# WinForge v5 — Windows Toolkit

A polished Windows optimization and repair toolkit. Clean up, tweak, debloat, and manage your system from one place.

---

## Features

| Tab | What it does |
|-----|-------------|
| System Info | Full hardware report — CPU, GPU (real VRAM), RAM, storage, security |
| Repair | DISM, SFC, CHKDSK |
| Cleanup | Temp files, Update cache, DNS flush, Network reset, Recycle Bin, Event logs |
| Dependencies | .NET 6/7/8/9, VC++ Redistributables, DirectX, WebView2, XNA |
| Install Apps | 28 apps across 7 categories, winget or Chocolatey |
| Tweaks & Debloat | Essential + Preference tweaks, Browser debloat, App removal |
| DNS Settings | Cloudflare, Google, Quad9, OpenDNS, AdGuard, Cloudflare Family, Auto |
| Win Updates | Default / Security Only / Disable |
| Registry Health | Scan & clean broken uninstall entries with automatic backup |
| System Tools | 15 Windows admin panels in one place |
| Restore Points | Create with 24h bypass, session-aware |

---

## Download
Get the latest `.exe` from the **Actions** tab → most recent build → **WinForge-v5** artifact.

## Requirements
- Windows 11
- Run as Administrator (app will prompt automatically)
