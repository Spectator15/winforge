# WinForge v4.0 — Windows Toolkit
Made by Danish

All tweaks use WinUtil-aligned backend commands for stability and correctness.

## Features
- System Repair (DISM, SFC, CHKDSK)
- Cleanup (grid layout)
- Dependencies (grid layout)
- Install Apps (winget/Chocolatey, search, 28 apps)
- Tweaks & Debloat (merged, Essential/Advanced/Preferences + app removal)
- DNS Settings (7 providers)
- Windows Updates (Default/Security/Disable)
- Registry Health (broken uninstall entries, empty keys, backup before clean)
- System Tools (legacy panels)
- Restore Points (24hr bypass, session-aware prompting)
